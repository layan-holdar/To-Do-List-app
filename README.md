# To-Do-List-app
A simple and interactive To-Do List application built with HTML, CSS and JavaScript
## Features
- Add a new task
- Mark tasks as completed
- Delete tasks
- Save tasks using local storage
## Technologies Used
- HTML
- CSS
- JavaScript
